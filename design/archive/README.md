# Archived visual assets

These files are preserved for design history and exploratory prototype reference. They are not approved production runtime assets.

- `runtime-home/` contains the former composited home tiles and village image.
- `earlier-directions/` contains superseded direction artwork.
- `earlier-screen-references/` contains earlier screen explorations and workups.
- `superseded-runtime-templates/` contains the previous classroom and notebook templates with baked controls.

Production code selects approved runtime artwork through `lib/production/assets.ts`.
