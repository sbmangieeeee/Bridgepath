# MVP Boundaries

> **Discovery status:** The implementation below records the existing prototype. Its Market Day context and world structure are not approved curriculum decisions. The first production content boundary will be selected after approval of `STORYPATH-DISCOVERY-BLUEPRINT-STANDARD-3-MATHEMATICS.md` and educator validation.

## Implemented vertical slice

- Responsive launch, role choice, parent consent/profile setup, family code and picture PIN.
- Playable orientation, Storypath Village, Market Day path, six learning stops.
- Optional browser narration, progressive feedback and remediation, local persistence.
- Derived learning status, coverage, support signals, recent evidence, export and erase.

## September 2026 production pilot

Add approved adult verification, live authentication and Supabase authorization, consent provenance, deletion workflows, production audio, reviewed illustration, monitoring, support, security/accessibility hardening, and 20–50-family cohort operations.

## Later

Expand Standard 3 Mathematics through the reusable activity/evidence contracts. Add Standards 4–5 and English after content-model validation. Defer teachers, diagnostics, native apps, free roaming, inventory, multiplayer, leaderboards, and advanced game systems.
